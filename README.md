# Infraestructura MinTic en AWS

Proyecto de infraestructura automatizada con CloudFormation y GitHub Actions.

## Arquitectura

- VPC con 2 subnets públicas
- Security Groups
- 2 Instancias EC2 estáticas
- Application Load Balancer
- Auto Scaling Group con 4 instancias

## Región

us-east-1 (Virginia)
